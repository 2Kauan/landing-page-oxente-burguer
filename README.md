A **Oxente Burguer** é uma landing page de alta conversão desenvolvida para uma hamburgueria artesanal baseada em Jaicós-PI. O projeto foi concebido para centralizar o fluxo de pedidos, migrando o atendimento manual do WhatsApp para uma interface digital otimizada, reduzindo o tempo de resposta e elevando o posicionamento da marca no mercado local.

---

## 🎯 Objetivo do Projeto

Transformar a presença digital da hamburgueria através de uma interface minimalista, moderna e com foco total em **Food Appeal**. O projeto elimina a fricção no atendimento inicial, oferecendo botões de ação rápida e um fluxo de decisão inteligente para o cliente.

## 🚀 Funcionalidades Principais

* **Fluxo de Decisão Estratégico:** Modal inteligente que segmenta o cliente entre *Delivery* e *Pedido na Mesa*, agilizando o processo antes mesmo da abertura do cardápio.
* **Design System Consistente:** Paleta de cores baseada em tons escuros (#0D0D0F) com acentos em Laranja Vibrante (#FF5E1A), mantendo coerência visual, estrutural e estratégica em todo o produto.
* **Performance Otimizada:** Desenvolvimento "Vanilla" (sem frameworks pesados), garantindo um carregamento instantâneo mesmo em conexões móveis instáveis.
* **Interface Mobile-First:** Totalmente responsivo e pensado para o uso com apenas uma mão, facilitando a experiência do cliente no smartphone.
* **SEO & Social Preview:** Integração com Meta Tags Open Graph para gerar cards profissionais e atraentes ao compartilhar o link no WhatsApp e Instagram.

## 🛠️ Tecnologias Utilizadas

* **HTML5 Semântico:** Estrutura focada em acessibilidade e indexação.
* **CSS3 Moderno:** Uso de variáveis nativas, Flexbox, CSS Grid e animações refinadas.
* **JavaScript Vanilla:** Manipulação dinâmica do DOM para controle de Modais e Menu Mobile sem dependências externas.
* **Tipografia:** Combinação estratégica das fontes *Fraunces* (Autoridade) e *DM Sans* (Leitura Confortável).

## 🎨 Identidade Visual

O design segue a filosofia **Premium Dark Mode**:
* **Primária:** `#FF5E1A` (Laranja Oxente) — Cor de ação para estimular o apetite e destacar CTAs.
* **Fundo:** `#0D0D0F` (Preto Profundo) — Para dar destaque absoluto às imagens dos lanches.
* **Texto:** `#EBEBF0` (Branco Acinzentado) — Garante contraste sem causar fadiga visual.
